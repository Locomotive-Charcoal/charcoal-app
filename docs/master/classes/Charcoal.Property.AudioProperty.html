setData()

setData(array  $data) : \Charcoal\Property\AbstractProperty

This function takes an array and fill the property with its value.

This method either calls a setter for each key (set_{$key}()) or sets a public member.

For example, calling with set_data(['ident'=>$ident]) would call setIdent($ident) becasue setIdent() exists.

But calling with set_data(['foobar'=>$foo]) would set the $foobar member on the metadata object, because the method set_foobar() does not exist.

array $data

Returns
